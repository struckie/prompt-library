# Prompt Templates

Reusable prompt templates organized by use case.

## Template Format

Each template should include:
- **Name:** Clear, descriptive title
- **Purpose:** What problem it solves
- **Usage:** How to use it
- **Variables:** What to customize
- **Example:** Working example

## Categories

- `work/` - Professional/Uniper prompts
- `personal/` - Homelab, learning, projects
- `meta/` - Prompts about prompting
